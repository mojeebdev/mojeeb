# Mojeeb | Systems • Strategy • Research

> "Longevity is the Ultimate Leverage."

An integrated dashboard for **Systems Architecture** and **Web3 Growth Strategy**. This repository houses the frontend for the [Mojeebeth Portfolio](https://mojeeb.xyz/), built with Next.js, Tailwind CSS, and Shadcn UI.

---

## 🛠 The Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS / Glassmorphism
- **UI Components:** Radix UI / Shadcn
- **Icons:** Lucide-React
- **State:** React Query / TanStack

## 🔬 Core Focus: Blindspot Labs
This site serves as the gateway to **Blindspot Labs**, a research initiative dedicated to the **2026 Series**. We focus on identifying structural failure points in Web3 ecosystems before they scale.

## 📈 Track Record
- **Taiku NFT:** Engineered viral growth from 3 → 9k followers in 3.5 days.
- **SkylosChain:** Optimized engagement systems from 5% to 95%.
- **Strategic Audits:** Ethereum, Base, Solana, Polygon, and Avalanche.
